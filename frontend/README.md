# Hospital Management System - Frontend

## Setup

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

-   **Authentication**: Login and Register for different roles.
-   **Patient Dashboard**: Book appointments, view medical history.
-   **Doctor Dashboard**: Manage appointments, prescribe meds, request lab tests.
-   **Lab Dashboard**: Upload test reports (simulates AI analysis).
-   **Pharmacy Dashboard**: Dispensary management.

## Tech Stack

-   React
-   Tailwind CSS
-   Vite
-   Axios
